# Luit School of Business
A futuristic, 3D animated web application built with React, Vite, and Tailwind CSS.

## Features
- **Scroll-Velocity Glitch Engine:** Chromatic aberration based on scroll physics.
- **Magnetic Physics Buttons:** Hover buttons with spring tracking.
- **Holographic HUD Elements:** Interactive SVG geometries and real-time telemetry.
- **Z-Space Holograms:** 3D perspective pop-outs for UI elements.
- **Pixel Assembling Graphics:** Images materialize pixel by pixel natively on scroll.

## Getting Started
1. `npm install`
2. `npm run dev`
